# Opinion Poll

A MERN based web application used to take opinions from the users on different task.

## Features
<ul>
    <li>1. Create Opinion</li>
    <li>2. Filter Opinions</li>
    <li>3. Support or Don't Support opinion</li>
    <li>4. Signup and login</li>
<ul>
